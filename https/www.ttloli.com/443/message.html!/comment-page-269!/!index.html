能，前提是你是免费分享，使用本人解压密码的请明确标明本人网站网址，除非使用你自己的解压密码，若收费下载（收费会员等方式）则禁止以任何形式转载，包括解压重新打包的，请自己下载镜像本体打补丁。我以前没作明确答复，导致各种复制站各种备份站各种转载站，分享gal资源不是敛财的手段，更不应该以各种什么服务器费用的理由而强制收费才能下载。
