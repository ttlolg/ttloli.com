能，前提是你是免费分享，使用本人解压密码的请明确标明本人网站网址，除非使用你自己的解压密码，若收费下载（收费会员等方式）则禁止以任何形式转载，包括解压重新打包的，请自己下载镜像本体打补丁。我以前没作明确答复，导致各种复制站各种备份站各种转载站，分享gal资源不是敛财的手段，更不应该以各种什么服务器费用的理由而强制收费才能下载。

能，一方面本站就是面向新人，另一方面本站不搞会员收费下载，网站和资源也不会添加任何个人广告，希望能让初接触galgame的伙伴能容易入坑和有个好的体验，对于什么圈子还是少一点人好、人多会变杂的言论，我还是不认同的，谁没有萌新的时候呢，让新人无头乱找与其让他有个好的导向，这个就是建站初衷！
三、访问网站相关
无法访问时请先打开https://mmgal.com，看看域名是不是已经变更。手机用户浏览器若出现什么政策无法访问的请关闭安全网址检测之类的设置，例如小米浏览器的在浏览器设置的隐私和安全里面。
