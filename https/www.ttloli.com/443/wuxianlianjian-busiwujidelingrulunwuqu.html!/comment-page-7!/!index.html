游戏简介

肩负着永恒的生命与严酷的命运的少女，通过自己的力量走向未来——
 
自2011年10月开始，开始进行3个月连续发表作品的Liquid。
其第二弹作品，是不老不死的少女横跨几个世纪的壮大及O辱的战斗物语。
一开始是奴隶的身份的女主角的生活，因为有着一副即使被杀也能立刻复活的身体，所以承受着极其残酷的凌辱。
即使是举起了反旗但也仍旧过着每天被追赶的艰辛的日子。
一边担心着追赶者一边在不同的时代生活下去，在各种各样的相遇与离别中越来越坚强的少女。
但是在没有尽头的逃避之路的前方，到底会有什么等待着她呢。。。
 
故事简介：
在由绝对的阶级制度所掌控的世界，当着劣等怪物的奴隶的主人公，过着每天经受精神上以及肉体上的淫辱的日子。
某天，主人公因为和世界之王产生共鸣所以被授予了不老不死的肉体的能力，但同时也被剥夺了一切的权利。
 
在主人公的身上实施过的猎奇性的残忍的凌虐暴行可以说是数之不尽。
像玩具一样地对待她那不死的身体，以此当做娱乐消遣的城市里的居民们。
因为是不老不死所以连自杀也办不到，所以主人公永远不可能会从痛苦的永劫中解放出来。
然后，得知了如果杀死王自己也随之死去的主人公为了终结自己的命运朝王以刃相向，但计划失败，沦落为被追赶之身逃离了城市。
 
然后经过了与许多人的邂逅以及离别……
时不时地遭到王手下的追赶者的胁迫的主人公过着仿佛无穷无尽般持续的时间。
 
到底，拥有永恒的生命的人的未来究竟会如何呢？

    个人评价：（有剧透）
    1.这部作品是完全披着拔作猎奇外皮的剧情作。
    2.舞美进入残酷的世界，里全是残酷杀戮，舞美的遭遇不能再惨，被触手怪当奴隶，被轮，被当公交（心里接受能力差的我差点玩不下去），但是这一点点的磨难都没有把舞美坚韧的内心磨灭。
    3.由于舞美的舞蹈打动的世界的王，王赐予了舞美“自由”，这个自由也就是自己不会被杀死，能无限复活。
    4.第二章开始剧情发生变化，舞美遇到了第一个关爱他，呵护她的吸血鬼伯爵，伯爵不在乎她悲惨肮脏的过去，并为了舞美所牺牲。舞美悲痛不已同时一步步的走向坚强。
    5.从第三张开始剧情渐入佳境，舞美在不同世界上遇见了更大的磨难，同时也经历了美好，一步步成长起来。
    6.剧情开始进入高潮，舞美在内泽瓦的提醒下发现原本世界的王就是获得不死能力几万年之后的自己，自己由于在世界末日孤独的一人，于是创造了舞美一开头那个残酷杀戮的世界。（不剧透更多，自己体味）
    7.本作品可以算作是一个轮回系的作品。
    综合评价：
    1.剧情有既视感，特别是男主的设定跟凉宫春日的忧郁中长门有希的作用颇为相似。
    2.关于剧情流程的视角问题，大部分都以女主为第一视角，男主前期存在感有点薄弱，这让习惯于普通gal的男主后宫视角的玩家可能感到不适应，更何况女主在第一章受到非人的待遇。心理承受能力差的根本玩不下去。
    3.关于演出方面有几个感人的地方其实可以做的更好，但鉴于是2011年的老游戏就不要求更多
    4.关于CV演出是本作一大亮点，舞美从开始的奴隶声线—感受到世界的温暖—慢慢变坚强、自信这一系列的声线变化让我颇为惊艳，属实不错。
    5.总体来说这部作品
    剧情A， 音乐B，声优演出A+ ，CG演出B（不喜欢被凌辱部分），剧情演出B+（有几处可以做的更好），系统A。
    适合喜欢猎奇、凌辱、渴望剧情的玩家体验。不适合纯爱，废萌玩家体验（本人就是）。剧情的优质弥补了我对于凌辱猎奇的反感。感叹一句galgame 的核心还是要剧情来支撑。
