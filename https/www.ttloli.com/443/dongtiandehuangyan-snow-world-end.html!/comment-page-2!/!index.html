――任何人都会说谎。

秋天，被卷入了老同伴所引起的骚动之中――
 
――于是，看到了初雪的今天。

但是，却能发觉这是在说谎。

一直，都觉得不太擅长。
一直，都认为双方是没有羁绊的。
一直，都认为是不该染指的。
 
但是――自己的真实情感，是无法背叛的。
 
“喜欢你――帝堂前辈。请和我交往吧”
 
比谁都要，……谎言的少年和
比谁都要，……谎言的少女的
 
宣告谎言终结的故事，就此开始了――
