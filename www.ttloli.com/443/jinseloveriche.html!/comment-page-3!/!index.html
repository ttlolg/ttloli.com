游戏简介

私立ノーブル学园。
培养未来的绅士和淑女的摇篮，是一所不仅教授教养，亦旨在培养品格的全住宿制学园。
 
然而今年，某个远方北欧小国的公主将要入学的消息，让学园感受到了不同往年的紧迫感。
 
就在这个时候，主人公市松央路由于陷入意想不到的事态而被那位公主所看中，误打误撞的闯入了这所学园。
也正在这个时候，没有空房间的缘故，他只好被塞进了女生宿舍最靠边的地方……

硬盘版制作流程

以下禁转载
点击展开查看

虽然制作硬盘版不难，但制作一个优质的还是要点心思的，后台很多评论回复感谢搬运，虽然不介意，但还是留给评论其他人吧。
制作了几年硬盘版，期间有不少人询问如何制作，所以之后的更新都会附带制作过程，还是看不会的话，最简单就是直接搬运别人做好的重新打包X_X
 
 
①使用虚拟光驱软件Daemon Tools加载日文版镜像文件，加载后打开虚拟光盘，先判断游戏制作引擎，可以看出这游戏是SiglusEngine引擎的，这引擎基本不需要运行setup.exe进行安装，直接复制到硬盘就可以运行。

 
②把游戏主目录复制到硬盘，然后新建个文件夹把汉化补丁解压出来。

 
③比对汉化补丁文件和日文版的文件有没有重复，经过比对，这作的汉化补丁是不覆盖任何文件的。

 
④把汉化补丁所有文件复制到游戏主目录下。

 
⑤新建编辑游戏说明档

 
⑥新建编辑汉化补丁卸载批处理脚本

 
⑦测试运行汉化版，可以正常运行。由于汉化补丁不覆盖日文版文件，可以在已经打了汉化补丁的情况下直接运行日文版，但发现日文版打不开，看提示是有区域限制。

 
⑧对于SiglusEngine引擎，大家可以寻找通用破解补丁AlphaROMdiE。也可以像我这样，打开反汇编软件Ollydbg，载入日文版程序，修改所有相关跳转

 
⑨修改完后，再运行测试，日文版能正常运行。

 
⑩最后是删除测试生成的存档，压缩打包上传。
注：保留原始汉化补丁压缩包是为了不让汉化补丁失传，若有人求这游戏的汉化补丁，请各位还是善于重上传分享。

    先评分，不算理亚线，给7分。单评理亚线，给6分及格分。加在一起，5分不能再高了。

    前排提醒，真结局有屎，想玩青春日常废萌的速退或者不要走最终的线路。

    先说一下玩着游戏的总体感觉吧，sage也是个游戏大厂了，甚至大名鼎鼎的初雪樱都是这个公司制作的，但是就在这一作上，我却感受到了各种既视感。
    首先是艾尔，头上的小猫耳头发，精致的饰品，眼睛睫毛的刻画方法，还有整体脸部神情，让我甚至一度激动的认为这是和泉老师负责的人设，但是作为一个优秀的和泉厨，我深知和泉老师没有参与这座的原画，即使如此也不影响我对这个角色产生了亲切感。而在剧情中，相当王道的剧情也让人感觉很舒服，没有波澜，就是单纯的轻松而已，所以我先通了这条线。
    其次是四加一的构成，四个金发女主，行吧，对应本作的名字，金色嘛，OK，有心了，但是这四加一中那加一的小茜线，这种如同存话费赠的一样的剧情让人不得不想起来柚子社的一贯做法，既视感+1
    再次是个人线剧情，简单的小茜线一股浓浓的八月废萌风，王女线让我想起了任性highspec的会长，玲奈……还行吧，但是也没有什么太吸睛的卖点。既视感+2

    也就是说，在玩这个游戏的时候，我总是感觉这个游戏是集业界之功力拼凑而成，当然，我也清楚这种同质化是不可避免的，所以此条并没有影响我的评分。

    进入正题。

    在玩完本作的时候，我就知道对于这部作品的评价一定是两极分化的，好的人觉得前面够废萌！hs够实用！理亚线感动了老子！所以好游戏！打差评的人会觉得：什么狗屁玩意？我玩的不是废萌？最后这条线是恶心我？去你妈的sage宣传欺诈！
    看到了么，最大的问题就是，本作前后冲突，存在宣传欺诈！你从任何的介绍中看到本作有悲情元素了吗？没有吧？没有！而且在玩家攻略的过程中，也没有感受到任何有要虐的迹象，甚至看起来最为艰难的王女线都没有面临身份地位带来的困难，一切都那么轻松，但是在最后一条线理亚线，突然来了个大转折，什么悲情催泪乱七八糟管它本来定位是什么，通通整起来！
    是的，对这种做法我是极其不耻的。

    也许sage认为，我前面搞废萌，再用双飞HS抓住读者，在读者对我的游戏产生思维定式的时候我来个大转折，打破这种定式，形成一种强烈的冲击，这部游戏的档次就会上升起来！隔壁钟表社euphoria就是这么干的，那我也来这么干！

    我感觉作为玩家我被戏耍了。

    好，不谈总体，单说前面的废萌部分，好玩吗？好玩，十分轻松搞笑的废萌作，天然呆的王女，正直的女骑士，善良亲切看起来辣妹其实很温柔的玲奈，还有热情活泼的学妹，剧情合格吗？作为废萌完全OK，有惊喜吗？有啊，突如其来的双飞，别管逻辑合不合理，当初我看到居然在主剧情里玩双飞我头都大了这这这搞什么啊！但是香不香呢？真特么香啊！尤其是黑长直，真是太棒了！除此之外，人设讨喜，妹子可爱，糖分充足，这都是优点，至于缺点，没有什么特别明显的缺点，总体来讲瑕不掩瑜。

    再单独说理亚线，想要升华主题，完全OK。但是其他线铺垫了吗？你可以说有，但我认为没有，理亚的表现在其他线里就像是背景板一样的人物，不管主角怎样，都是一副我祝福你呀，我帮助你啊这样的没什么特殊之处的态度，至于她的困境，抱歉，没有体现出来，而且这个不良的人设本身就让人不是很喜欢，尤其我是相当反感女生抽烟的，虽然后面有解释，但是我不买账，五感刺激可以通过各种方式，为什么非要抽烟？假设说，辣椒很奇怪，那薄荷不香吗？
    再说剧情本身，唉，铺垫不足啊，剧情起伏虽然有，但是完全不够优秀啊，这也是为什么我给了6分的原因，制作组为了让理亚线感人，逼玩家吃下这晚藏在饭下面的屎可谓是费劲了功夫，各种渲染刻画不一而足，然而即使如此，也没有达到任何优秀泣作的标准，只是在悲情剧情中还算合格。
    什么？你说本作是废萌，不是泣作？抱歉，我认为sage既然选择这样的结局，那么目的就是要脱离废萌范畴，要升华主题，那么单提出这条线来，我用泣作的标准来审视一点问题都没有。

    最后，sage对这个TE有没有后悔，sage从来没出过fd但是本作却出了一个理亚的另一个结局，什么意思我觉得已经不用解释了。

    总结，抛弃理亚线，是个优秀的废萌作。实用度高，双飞喜闻乐见，剧情轻松搞笑，妹子讨喜，值得一玩。加上理亚线，制作组想玩个骚操作给自己玩砸了。
